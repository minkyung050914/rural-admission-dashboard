# rural-admission-dashboard

# 농어촌 대입 전형 실효성 진단

교육 공공데이터 기반 교육소외지수(E_s) 시뮬레이션 대시보드

> 제8회 교육 공공데이터 AI 활용대회 제출작

## 프로젝트 개요

현행 농어촌 전형은 읍·면 행정구역 소재 여부만으로 대상을 판별합니다.
본 연구는 학원 밀도·교통접근성·학령인구·사교육비 등 6개 실질 변수로
**교육소외지수 E(s)** 를 산출하고, ML 역산으로 현행 기준의 한계를 정량화합니다.

🔗 **대시보드**: https://rural-admission-dashboard.streamlit.app/

## 핵심 결과

| 지표 | 수치 |
|---|---|
| 분석 대상 학교 | 5,702개교 |
| ML 역산 정확도 (AUC) | 0.894 |
| 허위 농어촌 발굴 | 51개교 |
| 카카오 API 커버리지 | 99.6% |

## 데이터 출처

| 변수 | 데이터 | 출처 | 라이선스 |
|---|---|---|---|
| α | 학원교습소 현황 | 교육부 data.go.kr | 공공누리 1유형 |
| β | 교통접근성 지표 (2023) | KOTI ktdb.go.kr | 공공누리 1유형 |
| γ | 연령별 주민등록인구 | 행안부 KOSIS | 공공누리 1유형 |
| δ | 사교육비·참여율 | 교육부·통계청 | 공공누리 1유형 |
| - | 학교 위치 정보 | 한국지방교육행정연구재단 | 공공누리 1유형 |
| - | 법정동 코드 | 행안부 code.go.kr | 공공누리 1유형 |
| - | 좌표→행정동 변환 | 카카오 Local API | 카카오 개발자 약관 |

## 분석 파이프라인

Step 1: 학교 마스터 + 현행 농어촌 라벨링 (읍·면 여부)
Step 2: 전국 학원 17개 CSV 병합 → 시군구 단위 α 변수
Step 3: KOTI 교통접근성 + 카카오 API → β 변수
Step 4: 주민등록인구 행정동 JOIN → γ 변수
Step 5: 사교육비 시도·학교급 매핑 → δ 변수
Step 6: Min-Max 정규화 → E(s) 산출 → ML 역산 분석

## 실행 방법

pip install -r requirements.txt

| β₁ | 대중교통 광역교통시설 최단 접근시간 | 낮을수록 도시형 |
| β₂ | 30분 내 광역교통시설 수 | 높을수록 도시형 |
| γ₁ | 읍면동 10~19세 주민등록인구 | 높을수록 도시형 |
| δ₁ | 시도·학교급 월평균 사교육비 | 높을수록 도시형 |
| δ₃ | 시도·학교급 사교육 참여율 | 높을수록 도시형 |

현행 읍·면 기준이 암묵적으로 부여하는 가중치:

| 변수 | 로지스틱 회귀 | 랜덤 포레스트 | 평균 |

| β₁ 교통시간 | 0.421 | 0.273 | **0.347** |
| δ₁ 사교육비 | 0.354 | 0.165 | **0.259** |

→ 현행 기준은 δ₃(참여율), γ₁(학령인구)를 사실상 무시하고 있음
